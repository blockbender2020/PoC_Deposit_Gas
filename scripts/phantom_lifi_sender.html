Phantom Solana LI.FI Sender

Paste either the whole backend intent JSON or just the nested LI.FI quote JSON. This helper looks for a serialized Solana transaction payload and asks Phantom to sign and send it.

This helper is Solana-only. Use it for source-side Solana funding flows. After Phantom returns a signature, paste that signature back into the terminal wizard or submit it to the backend as sourceTxId.
